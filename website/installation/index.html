Install env-sync

Get peer-to-peer secret synchronization running in under a minute. One command, no accounts, no server to provision.

Quick install (recommended)

# system-wide install
curl -fsSL https://envsync.arnav.tech/install.sh | sudo bash

# user-only install (no sudo)
curl -fsSL https://envsync.arnav.tech/install.sh | bash -s -- --user

The installer downloads prebuilt binaries for your platform and handles service stop/restart during upgrades.

Prerequisites

  • Go 1.24+ — only needed if building from source
  • SSH client — for trusted-owner-ssh mode (installed by default on most systems)
  • mDNS support — Avahi on Linux, Bonjour on macOS (built-in), relay in WSL2

Build from source

git clone https://github.com/championswimmer/env.sync.local.git
cd env.sync.local
make build
make test
sudo make install        # system-wide
# or: make install-user  # user-only

Platform notes

Linux

sudo apt-get install -y avahi-daemon avahi-utils

Avahi provides mDNS peer discovery. Most desktop Linux distributions include it by default.

macOS

Bonjour is built-in — no extra packages needed. Service management uses launchd.

Windows (WSL2)

Run inside WSL2 or configure an Avahi reflector. Native Windows is not supported yet.

Post-install setup

# verify installation
env-sync --version

# initialize and start syncing
env-sync init
env-sync status
env-sync discover

# optional: set up automatic sync
env-sync cron --install

Upgrade & uninstall

# upgrade — re-run the installer
sudo ./install.sh

# uninstall
env-sync service uninstall
rm -rf ~/.config/env-sync
sudo rm -f /usr/local/bin/env-sync
