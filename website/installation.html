Installation guide

Prerequisites

  • Go 1.24+ (only for source builds)
  • SSH client for trusted-owner-ssh mode
  • mDNS support: Avahi on Linux, Bonjour on macOS, relay/workaround in WSL2

Quick install (recommended)

# system install
curl -fsSL https://raw.githubusercontent.com/championswimmer/env.sync.local/main/install.sh | sudo bash

# user install
curl -fsSL https://raw.githubusercontent.com/championswimmer/env.sync.local/main/install.sh | bash -s -- --user

Installer can download prebuilt releases and handles service stop/restart during upgrade.

Build from source

git clone https://github.com/championswimmer/env.sync.local.git
cd env.sync.local
make build
make test
sudo make install  # or make install-user

Platform notes

Linux

sudo apt-get install -y avahi-daemon avahi-utils

macOS

Bonjour is built-in. Service management uses launchd.

Windows (WSL2)

Prefer running on Windows host, or configure Avahi reflector in WSL2.

Post-install setup and verification

env-sync --version
env-sync mode get
env-sync init
env-sync status
env-sync discover
env-sync cron --install

Upgrade / uninstall

# upgrade
sudo ./install.sh

# uninstall
env-sync service uninstall
rm -rf ~/.config/env-sync
sudo rm -f /usr/local/bin/env-sync
