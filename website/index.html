Distributed .env sync across local peers

env-sync is a peer-to-peer secrets synchronization tool for local networks. No master server, automatic mDNS discovery, and explicit security models for different trust boundaries.

Go 1.24+ mDNS discovery SSH / HTTPS+mTLS AGE encryption

Get started Explore modes 
# install
curl -fsSL https://raw.githubusercontent.com/championswimmer/env.sync.local/main/install.sh | bash -s -- --user

# initialize and sync
env-sync init
env-sync add OPENAI_API_KEY="sk-..."
env-sync sync

Architecture summary

  • Distributed and eventually consistent: shell startup, cron, or manual sync.
  • mDNS discovery on local networks (Avahi/Bonjour).
  • Backup before overwrite (keeps last 5 copies).
  • Mode-aware sync orchestration in Go.

Core workflows

  • Trusted-owner: discover peers → SCP/SSH fetch → compare → merge → save.
  • Secure-peer: discover → mTLS auth → authorization check → decrypt/merge → re-encrypt to recipients.
  • Peer lifecycle: invite → request-access → approve/revoke → signed membership propagation.
